# sistema-pagos-playwright-tests
Pruebas de regresiòn sobre Sistema de Pagos


Framework de automatización E2E para un sistema de pagos desarrollado en .NET.

El proyecto incluye:
• Playwright para pruebas de regresión
• SQL Server ejecutado en Docker
• Docker Compose para levantar la aplicación completa
• GitHub Actions para ejecutar pruebas automáticamente en CI

Las pruebas cubren flujos críticos como login, gestión de pagos, tipos de gasto y usuarios.

El proyecto incluye:
• Playwright para pruebas end-to-end
• Docker y SQL Server para levantar el entorno completo
• GitHub Actions para ejecutar las pruebas en CI
